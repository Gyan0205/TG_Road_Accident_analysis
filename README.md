# Road Accident Risk Analysis using Power BI  
### Telangana Case Study

## 📌 Project Overview
This project focuses on analyzing road accident patterns using an interactive Power BI dashboard.  
The objective is to identify accident hotspots, peak accident hours, severity distribution, and environmental risk factors to support data-driven road safety planning.

## 🎯 Problem Statement
Road traffic accidents are a major public safety issue resulting in injuries, fatalities, and economic losses.  
Traditional static reports make it difficult to identify high-risk locations and contributing factors.  
There is a need for an analytical system to explore accident trends interactively.

## 💡 Proposed Solution
An interactive Power BI dashboard was developed to visualize accident data and generate actionable insights.  
The dashboard enables multi-level analysis including geographical distribution, temporal trends, and risk factor evaluation.

## 🛠 Technologies Used
- Power BI Desktop  
- Power Query (Data Cleaning & Transformation)  
- DAX (Calculated Measures & KPIs)  
- Excel / CSV Dataset  

## 🔄 Data Preprocessing Steps
- Removal of irrelevant columns  
- Handling missing and coded categorical values  
- Conversion of numerical codes into descriptive labels  
- Filtering accident data for Telangana region  
- Creation of calculated columns such as Day Name and Severity Category  

## 📊 Dashboard Features
- Accident hotspot visualization using map  
- Peak accident hour analysis  
- Weekly accident trend analysis  
- Accident severity distribution  
- Weather and lighting condition impact  
- Road surface and speed limit risk analysis  
- Urban vs Rural accident comparison  

## 📈 Key Insights
- Accident density is higher in urban regions and transport corridors  
- Peak accidents occur during evening traffic hours  
- Majority of accidents are low to medium severity  
- Fine weather conditions record higher accident frequency  
- Higher speed zones show increased accident severity  

## ✅ Conclusion
The project demonstrates how business intelligence tools can support road safety analysis through interactive visualizations and insight-driven decision making.

## 🚀 Future Scope
- Integration of real-time accident data  
- Machine learning based accident prediction  
- Expansion to nationwide accident monitoring dashboards
